About Me

Hello! I'm Mahfuz Ahammed, a passionate Software Engineer from Bangladesh with professional experience in building scalable web applications using ASP.NET Core, Angular, SQL Server, and modern software development practices.

I enjoy solving real-world business problems through software solutions and continuously learning new technologies to improve my technical and professional skills.

Professional Experience

Currently, I am working as a Software Engineer at Solution World Ltd., where I develop enterprise-level applications using ASP.NET Core Web API, Angular, TypeScript, SQL Server, and Git.
